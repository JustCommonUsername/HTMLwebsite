  Для начала, важно рассказать, что же такое HTML.

HTML (от англ. HyperText Markup Language — «язык гипертекстовой разметки») — HTML (или XHTML). Язык HTML интерпретируется браузерами; полученный в результате интерпретации форматированный текст отображается на экране монитора компьютера или мобильного устройства.

Теперь, когда мы ознакомились с HTML, я представляю вам теги, благодаря которым был создан этот сайт.

Теги и их функции в HTML
Teги Описание их функций
<! --...--> Используется для добавления комментариев.
<!DOCTYPE> Объявляет тип документа и предоставляет основную информацию для браузера — его язык и версия.
<head>...</head> Элемент-контейнер для метаданных HTML-документа, таких как <title>, <meta>, <script>, <link>, <style>.
<body>...</body> Представляет тело документа (содержимое, не относящееся к метаданным документа).
<html>...</html> Корневой элемент HTML-документа. Сообщает браузеру, что это HTML-документ. Является контейнером для всех остальных html-элементов.
<title>...</title> Заголовок HTML-документа, отображаемый в верхней части строки заголовка браузера. Также может отображаться в результатах поиска, поэтому это следует принимать во внимание во время предоставление названия.
<meta> Используется для хранения дополнительной информации о странице. Эту информацию используют браузеры для обработки страницы, а поисковые системы — для ее индексации. В блоке <head> может быть несколько тегов <meta>, так как в зависимости от используемых атрибутов они несут разную информацию.
<pb /> Переносит текст на следующую строку.
<p> Параграфы в тексте.
<h1 - h6>...</h1 - h6> Создают заголовки шести уровней для связанных с ними разделов.
<table>...</table>
  • <tr>...</tr>
  • <th>...</th>
  • <td>...</td>
 Тег для создания таблицы.
  • Создает строки(!) таблицы.
  • Создает заглавную ячейку таблицы. Помните о том, что тег <th> играет не просто визуальную роль, а семантическую. Это важно для поисковых роботов.
  • Создает ячейку таблицы.
